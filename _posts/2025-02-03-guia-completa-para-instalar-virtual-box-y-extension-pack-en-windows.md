### Instalación de VirtualBox y Extension Pack
El siguiente manual detalla el proceso para instalar VirtualBox y su Extension Pack en un entorno Windows. Esta herramienta es esencial para crear máquinas virtuales y configurar laboratorios de sistemas operativos, facilitando la experimentación y el aprendizaje.

1. Descarga del instalador de VirtualBox

    Para obtener el instalador, navega al sitio oficial de VirtualBox: virtualbox.org.     
    Una vez en la página principal, haz clic en 'Download' para acceder a la sección de descargas.

    ![alt text](/assets/img/linux-003-1.webp){: width="700" }

    Selecciona el sistema operativo 'Host' (anfitrión) para tu máquina virtual. En este caso, haremos clic en 'Windows hosts' para iniciar la descarga del instalador.

    ![alt text](/assets/img/linux-003-2.webp){: width="700" }

2. Proceso de instalación de VirtualBox

    Cuando la descarga finalice, ejecuta el archivo (.exe) con doble clic para iniciar el asistente de instalación.
    ![alt text](/assets/img/linux-003-3.webp){: width="500" }

    En la ventana principal del asistente, haz clic en 'Next'.
    ![alt text](/assets/img/linux-003-4.webp){: width="450" }

    Acepta los términos de la licencia de uso y haz clic en 'Next'.
    ![alt text](/assets/img/linux-003-5.webp){: width="450" }

    Mantén la configuración de instalación predeterminada y haz clic en 'Next'.
    ![alt text](/assets/img/linux-003-6.webp){: width="450" }

    Confirma el inicio de la instalación haciendo clic en 'Yes'.
    ![alt text](/assets/img/linux-003-7.webp){: width="450" }

    VirtualBox podría solicitar la instalación de dependencias adicionales para su correcto funcionamiento. Acepta este paso haciendo clic en 'Yes'.
    ![alt text](/assets/img/linux-003-8.webp){: width="450" }

    Espera a que la instalación se complete.
    ![alt text](/assets/img/linux-003-9.webp){: width="450" }

    ![alt text](/assets/img/linux-003-10.webp){: width="450" }

    Al finalizar, desmarca la opción para iniciar el programa de inmediato. Esto nos permitirá instalar primero el paquete de extensiones, el cual es necesario para funcionalidades como el soporte de dispositivos USB y el cifrado de discos.
    
    ![alt text](/assets/img/linux-003-11.webp){: width="450" }

3. Instalación del VirtualBox Extension Pack

    En la página de descargas de VirtualBox, el paquete de extensiones está disponible bajo el título 'VirtualBox 7.0.xx Oracle VM VirtualBox Extension Pack'. Haz clic en 'All supported platforms' o 'Accept and Download' para obtenerlo.
    ![alt text](/assets/img/linux-003-12.webp){: width="700" }

    Una vez descargado, ejecuta el archivo (.vbox-extpack) con doble clic. Esto abrirá automáticamente VirtualBox.
    ![alt text](/assets/img/linux-003-13.webp){: width="500" }

    En la ventana emergente, haz clic en 'Instalar' para continuar.
    ![alt text](/assets/img/linux-003-14.webp){: width="700" }

    Lee y acepta los términos de la licencia del paquete de extensiones haciendo clic en 'Acepto'.
    ![alt text](/assets/img/linux-003-15.webp){: width="700" }

    Una vez finalizada la instalación del Extension Pack, la herramienta estará lista para la creación y gestión de máquinas virtuales.
    ![alt text](/assets/img/linux-003-16.webp){: width="700" }
